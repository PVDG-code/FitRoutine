PLACEHOLDER ASSETS

When you run the Expo app for the first time, you may see warnings about missing icons.
This is normal - Expo will use default placeholders.

To add custom icons later (optional):
1. Create icon.png (1024x1024)
2. Create splash.png (2048x2048)
3. Create adaptive-icon.png (1024x1024) for Android
4. Create favicon.png (48x48) for web

Place them in this assets folder.

For now, the app will work perfectly fine with Expo's default placeholders!
